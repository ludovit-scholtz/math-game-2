# Google Play ストア用テキスト - 日本語

## アプリ名

子ども向け算数練習 | Biatec

## 短い説明

たし算、ひき算、かけ算、わり算を楽しく時間制限つきで練習できます。

## 詳しい説明

Math Masterは、子どもが4つの基本的な計算である、たし算、ひき算、かけ算、わり算に自信を持てるようにするアプリです。100未満の数を練習する低学年の学習者向けに作られていますが、日常の算数を短時間でやさしく復習したい人にも役立ちます。

ゲームはシンプルで集中しやすく、やる気が続くように作られています。プレイヤーを選び、チャレンジ時間を決め、練習する計算を選んで、時間が終わるまでにできるだけ多くの問題に答えます。各ラウンドではすぐにフィードバックが表示され、正解をほめ、間違いもやさしく示すので、子どもがプレッシャーなく上達できます。

子どもに毎日の算数練習をさせたい保護者にとって、Math Masterは余計な気を散らす要素の少ない、すっきりした学習体験を提供します。学習、速さ、正確さ、反復に集中できます。間違えた問題はあとで再び出てくるため、同じ考え方にもう一度出会い、練習を通じて記憶を強められます。

主な機能:

- たし算、ひき算、かけ算、わり算を練習。
- すべての数は100未満で、初期学習者に適しています。
- 1分、2分、5分のチャレンジを選択。
- 1つの計算だけ、または複数の計算を同じラウンドで練習。
- 6つの答え候補で、各問題をすばやく簡単にプレイ。
- よくある間違いをもとにした不正解候補で、間違いから学べます。
- ゲーム履歴で、各プレイヤーが何を遊び、何回間違えたかを確認。
- ローカルランキングがゲームタイプごとのベストスコアを保存。
- 同じ端末で複数のプレイヤープロフィールを使用可能。
- 各プレイヤーが自分の言語設定を保存できます。
- 各プレイヤーはネコ、イヌ、パンダ、ウサギのペットを選び、コインでお世話できます。
- コイン、ショップアイテム、ボタンのカスタマイズがほどよい動機づけになります。
- 効果音が正解をほめ、間違いを知らせます。

Math Masterは子どもにやさしく、プライバシーにも配慮しています。プレイヤー名は簡単なニックネームでよく、進み具合は端末内にローカル保存されます。練習を始めるためにアカウントは必要ありません。

学校で算数を学んでいる子、暗算をもっと速くしたい子、毎日数分だけ集中して練習したい子に、Math Masterは基本計算を楽しく繰り返せる学習習慣に変えてくれます。
