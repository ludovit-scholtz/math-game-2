# Google Play 스토어 문구 - 한국어

## 앱 이름

어린이 수학 연습 | Biatec

## 짧은 설명

덧셈, 뺄셈, 곱셈, 나눗셈을 재미있게 시간 제한으로 연습하세요.

## 전체 설명

Math Master는 아이들이 네 가지 기본 수학 연산인 덧셈, 뺄셈, 곱셈, 나눗셈에 자신감을 갖도록 도와줍니다. 100 이하의 수를 연습하는 어린 학습자를 위해 특별히 설계되었지만, 일상 산수를 빠르고 부담 없이 복습하고 싶은 누구에게나 유용합니다.

게임은 단순하고 집중하기 쉬우며 동기를 줍니다. 플레이어를 고르고, 도전 시간을 선택하고, 연습할 연산을 고른 뒤 시간이 끝나기 전까지 최대한 많은 문제에 답하세요. 각 라운드는 즉시 피드백을 주고, 정답을 보상하며, 실수를 부드럽게 보여 주어 아이들이 부담 없이 나아질 수 있게 합니다.

자녀에게 규칙적인 수학 연습을 제공하고 싶은 부모에게 Math Master는 산만함이 적은 깔끔한 학습 경험을 제공합니다. 학습, 속도, 정확도, 반복 연습에 집중합니다. 틀린 문제는 나중에 다시 나와 같은 개념을 다시 만나고 연습으로 기억을 강화하도록 돕습니다.

주요 기능:

- 덧셈, 뺄셈, 곱셈, 나눗셈 연습.
- 모든 숫자는 100 미만으로, 어린 학습자에게 적합합니다.
- 1분, 2분, 5분 도전 선택.
- 한 가지 연산만 연습하거나 여러 연산을 한 라운드에 섞어 연습.
- 여섯 개의 답 선택지로 빠르고 쉽게 플레이.
- 오답은 흔한 실수를 바탕으로 만들어져 학습을 돕습니다.
- 게임 기록에서 각 플레이어가 무엇을 했고 실수가 몇 번이었는지 확인.
- 로컬 순위표가 게임 유형별 최고 점수를 저장.
- 같은 기기에서 여러 플레이어 프로필 사용 가능.
- 각 플레이어가 자신의 언어 설정을 유지할 수 있습니다.
- 각 플레이어는 고양이, 강아지, 판다, 토끼 중 하나를 선택하고 코인으로 돌볼 수 있습니다.
- 코인, 상점 아이템, 버튼 꾸미기가 부드러운 동기를 더합니다.
- 효과음이 정답을 보상하고 실수를 알려 줍니다.

Math Master는 어린이 친화적이며 개인정보를 존중합니다. 플레이어 이름은 간단한 별명으로 충분하고, 진행 상황은 기기에 로컬로 저장됩니다. 연습을 시작하는 데 계정이 필요하지 않습니다.

아이가 학교에서 산수를 배우고 있든, 더 빠른 암산을 준비하든, 매일 몇 분의 집중 연습이 필요하든, Math Master는 기본 연산을 즐겁고 반복 가능한 학습 습관으로 바꿔 줍니다.
